# 🐧 Руководство по Linux

Добро пожаловать в мир Linux! Здесь собраны основные команды и приёмы для эффективной работы в терминале. 🚀

## ⏰ Планировщик задач

### Редактировать задачи текущего пользователя
```bash
crontab -e
```   
### Показать задачи
```bash
crontab -l
```
### Удалить все задачи  
```bash
crontab -r    
```

**📐 Формат строки:**
```
# ┌──── минута (0–59)
# │ ┌── час (0–23)
# │ │ ┌ день месяца (1–31)
# │ │ │ ┌ месяц (1–12)
# │ │ │ │ ┌ день недели (0–6, Вс=0)
# │ │ │ │ │
# * * * * * команда
```
**📌 Примеры:**
```bash
0 2 * * * /usr/bin/backup.sh                 # 📅 ежедневно в 02:00
*/15 * * * * /usr/local/bin/check.sh        # ⏱️ каждые 15 минут
0 3 * * 0 /usr/bin/apt update && apt upgrade -y  # 📆 воскресенье, 03:00
```

### ⏳ Systemd Timers (современная альтернатива)
```bash
systemctl list-timers --all
systemctl status myjob.timer
journalctl -u myjob.service -f
```

## 🔍 Поиск больших файлов (>30 МБ)
```bash
find / -type f -size +30M -exec ls -lh {} + 2>/dev/null | sort -k5 -h
```
- `find /` — поиск от корня  
- `-type f` — только файлы  
- `-size +30M` — >30 мегабайт  
- `2>/dev/null` — подавить ошибки доступа  
- `sort -k5 -h` — сортировка по 5-му полю (размер) в человекочитаемом формате

### 🛡️ Настройка sudo без пароля для пользователя alice

Чтобы разрешить пользователю `alice` использовать `sudo` **без ввода пароля**, нужно отредактировать файл `/etc/sudoers` с помощью утилиты `visudo`. Это безопасный способ редактирования, так как он проверяет синтаксис перед сохранением.

## 🚶 Работа с пользователями и паролями шаги:

1. Откройте файл `/etc/sudoers` с помощью команды:
   ```bash
   sudo visudo
   ```

2. Добавьте в конец файла (или в соответствующее место) следующую строку:
   ```bash
   alice ALL=(ALL) NOPASSWD: ALL
   ```

   Эта строка означает:
   - `alice` — имя пользователя;
   - `ALL` — с любого хоста;
   - `(ALL)` — может выполнять команды от имени любого пользователя;
   - `NOPASSWD: ALL` — без запроса пароля для любых команд.

3. Сохраните и выйдите из редактора:
   - Если используется `nano`: `Ctrl+O`, `Enter`, затем `Ctrl+X`.
   - Если используется `vi`/`vim`: нажмите `Esc`, затем введите `:wq` и `Enter`.

## Альтернатива: через файл в `/etc/sudoers.d/`

Можно создать отдельный файл для пользователя `alice`:

```bash
sudo visudo -f /etc/sudoers.d/alice-nopasswd
```

И добавить туда ту же строку:
```bash
alice ALL=(ALL) NOPASSWD: ALL
```

Затем сохраните и выйдите.

> ⚠️ **Важно**: Убедитесь, что права доступа к файлу корректны:
> ```bash
> sudo chmod 440 /etc/sudoers.d/alice-nopasswd
> ```
